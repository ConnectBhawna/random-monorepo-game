# How to run the project:

1. Clone the project
   - git clone https://github.com/rohitranjan-2702/random-monorepo-game.git
2. Install dependencies
   - pnpm install
3. Run the project
   - pnpm build
   - pnpm dev
4. Open the browser and navigate to http://localhost:3000
5. You are ready to go!
